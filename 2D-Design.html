Course Description

Exploration of different 2D mediums, using cut paper. ColorAid and paints.

Found Line Collages

The goal of this project was to experiment with found lines and how they can interact with one another. The first one was made with a cool or quite color pallet in mind while the second has warmer and louder colors.

3 Notans

In this one I explored silhouette and patterns of reflection. Each piece reflects out from the original square in a different way giving each piece its own unique mood.
